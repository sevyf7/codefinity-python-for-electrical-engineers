# Sensor Interface Designer

This Python project designs a voltage divider sensor interface using standard resistor values.

## Features
- Calculates resistor values for a voltage divider
- Uses the voltage divider equation
- Selects the closest standard E12 resistor values
- Computes the actual output voltage after resistor approximation

## Concepts Used
- Voltage dividers
- Sensor interfacing
- Standard resistor series (E12)
- Circuit analysis
- Engineering approximation

## Technologies
- Python

## Example
Input:
- Vin = 5V
- Desired Vout = 3.3V

Output:
- Recommended resistor values
- Actual output voltage
