# Simple Circuit Analyzer

This Python project analyzes a simple electrical circuit using Python.

## Features
- Calculates total resistance
- Calculates total current
- Calculates branch currents
- Uses Ohm's Law
- Uses series and parallel resistance calculations

## Concepts Used
- Ohm's Law
- Series circuits
- Parallel circuits
- Basic circuit analysis

## Technologies
- Python

- Multi-loop circuit solving
